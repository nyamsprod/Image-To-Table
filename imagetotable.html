Converting an image into a table using the canvas tag

In order to convert and image into a table you browser needs to support

  • the fileReader API
  • the canvas API
  • Pixel Manipulation with Canvas

Or drag and drop a file here
To get the best result use small images or your browser may freeze or die on you!!

Without uploading the file to my server I can render it using a html table (with some help from the canvas tag)
